# simplePages
A simple theme based on the Primer theme for github Pages

## Condfiguration:

### Config file
File `docs/_config.yml`:
```yml
remote_theme: HuguesGuilleus/simplePages
```

### Fichier 404
File: `docs/404.md`:
```
---
permalink: /404.html
layout: e404
---
```

### Fichier root
File: `docs/index.html`:
```
---
layout: root
---
```
